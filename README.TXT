PROJECT TITLE: ConnecTextFour
PURPOSE OF PROJECT: A text-based implementation of the classic Connect Four game where two players take turns dropping pieces into columns to get four in a row.
VERSION or DATE: Version 1.0 (December 12, 2015)
HOW TO START THIS PROJECT: 
    1. Compile both ConnecTextFour.java and Texel.java
    2. Run: java ConnecTextFour
    3. Follow the on-screen prompts to play
AUTHORS: Brice Vollmer
USER INSTRUCTIONS: 
    - Two players alternate turns (Player T and Player B)
    - Enter a column number (1-9) when prompted
    - Your piece will drop to the lowest available row in that column
    - Win by getting 4 pieces in a row (horizontal, vertical, or diagonal)
    - Game ends when someone wins or a column becomes full
    - Invalid moves will prompt you to try again

GAME FEATURES:
    - 7 rows x 9 columns game board
    - Dynamic board representation using Texel objects
    - Input validation and error handling
    - Win detection for all directions
    - Text-based interface for easy accessibility
    - Replay option at the end of each game
